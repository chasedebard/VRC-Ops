# End User License Agreement

**Last updated:** July 13, 2026

VRC Ops does not currently provide a custom end user license agreement for its iOS, iPadOS, or macOS applications. The license to those applications is governed by Apple’s **Standard Licensed Application End User License Agreement**:

[Read Apple’s Standard EULA](https://www.apple.com/legal/internet-services/itunes/dev/stdeula/)

Apple states that its Standard EULA applies automatically when an app provider does not supply a custom EULA. The Standard EULA governs the license to the Apple-platform app software. The separate [VRC Ops Terms of Use](TERMS_OF_USE.md) govern VRC Ops accounts, leagues, online services, content, and subscriptions.
